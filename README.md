# K-Means聚类分析广告效果

**项目背景**：

为优化多元渠道的广告投放效果，本项目基于90日运营数据，运用聚类分析对广告渠道进行智能分组。通过深度挖掘每类渠道的核心特征（如高引流、高转化或高曝光），为制定差异化的投放策略、实现广告预算的精准分配和效果优化，提供明确的数据依据和决策支持。

**项目介绍**：

1. 本项目使用了Python进行K-Means聚类和PCA降维，PCA降维是为了后续的可视化
2. 使用Plotly和Seaborn对其进行雷达图和散点图可视化，以便分析不同渠道类别的显著特征。
